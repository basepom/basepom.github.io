Documentation for plugins and other components that are part of the basepom.org effort.

Apache Maven Plugins

  • Inline Plugin - inline dependencies into a main artifact.
  • Dependency Versions Check Plugin - verifies that the resolved versions of project dependencies are mutually compatible to each other.
  • Duplicate Finder Plugin - Finds and flags duplicate and conflicting classes and resources on the java classpath.
  • Property Helper Plugin - A maven plugin to generate and manipulate properties during the build process.
  • Repack Plugin - A maven plugin to repack code and its dependencies into a single archive.
